#Introduction

Welcome to Battleships where you can play against an AI and conquer victory against it!

#Prerequisities

1. Python 3.12 installed on your system
2. Access to a web browser
3. Access to the folders 'tests' and 'templates'
4. Access to the text files 'battleships' and 'placement'
5. Access to the json file 'player_placement'

#Installations

1. Open your command prompt or terminal and run the command 'pip install flask'
2. Open your command prompt or terminal and run the command 'pip install pytest'

#Testing

Run the pytests that are provided in the folder named 'tests'

#How to play the game

1. Run 'main.py'
2. To place your battleships go to http://127.0.0.1:5000/placement
3. After finishing placing your battleships, click 'send board'
4. You will then be taken to the main game interface
5. Attack the AI's battleships by clicking anywhere on its board
6. Enjoy the game!
